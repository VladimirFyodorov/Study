## Task #2 - Объединения (Union Types)
Реализуйте функцию lastIndex(str, char), возвращающую индекс последнего вхождения символа в строку или null, если такого символа нет.
```
const str = 'test';
lastIndex(str, 't'); // 3
lastIndex(str, 'p'); // null
```
